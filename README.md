Codex Android App
Overview
The Codex Android App is a feature-rich application tailored for Android devices, providing users with an immersive experience in exploring various content and functionalities offered by Codex.

Features
User-friendly Interface: Intuitive design for easy navigation.
Content Exploration: Access to a wide range of content categorized neatly.
Offline Access: Ability to view saved content without an internet connection.
Push Notifications: Stay updated with the latest content and updates.
Getting Started
Prerequisites
Android Studio (latest version)
Basic understanding of Kotlin and Android development
Installation
Clone the repository:
git clone https://github.com/mandarab76/codex-android-app.git
Open the project in Android Studio.
Build the project to install necessary dependencies.
Running the App
Connect your Android device or use an emulator.
Click on the Run button in Android Studio to start the application.
